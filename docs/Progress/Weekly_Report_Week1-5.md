## 1. 项目概述

### 1.1 项目背景
个性化旅游推荐系统是一个面向数据结构课程设计要求的综合应用系统，旨在为用户提供个性化的旅游推荐、路线规划、设施查询、美食推荐、日记管理与后台管理等功能。项目重点在于数据结构与算法在业务中的落地能力，必须遵循课程设计的硬性约束。

### 1.2 技术栈
- **前端**：Vue 3 + TypeScript + Vite + Pinia + Element Plus + ECharts
- **后端**：Spring Boot 3.2.x + MyBatis-Plus + MySQL + Redis(可选)
- **鉴权**：JWT + Spring Security
- **核心约束**：
  - 核心算法必须自主实现
  - 核心数据结构模块不得直接使用Java内置集合实现（需使用自定义结构）
  - 搜索/检索不得在SQL层完成，必须采用"数据库预加载 -> 内存索引 -> 算法检索"

### 1.3 项目里程碑
- **M1（第2周）**：需求与技术设计完成
- **M2（第4周）**：核心功能框架实现
- **M3（第5周）**：核心算法与真实数据集成完成

### 7.2 技术亮点

1. **自定义数据结构实现**：
   - ArrayList、LinkedList、HashMap、HashSet、PriorityQueue等
   - 符合课程设计要求，不直接使用Java内置集合

   - 前缀Trie树索引（精确前缀匹配）
   - NGram倒排索引（模糊全文检索）
   - 符合"数据库预加载 -> 内存索引 -> 算法检索"约束

3. **图算法应用**：
   - Dijkstra最短路径算法自主实现
   - 支持多种权重策略（距离、时间）
   - 支持多点路径规划

4. **真实数据集成**：
   - OSM数据采集脚本开发
   - Nominatim + Overpass API集成
   - 真路网生成和POI吸附

5. **前后端分离架构**：
   - RESTful API设计
   - JWT无状态认证
   - Vue 3 + TypeScript现代化前端
